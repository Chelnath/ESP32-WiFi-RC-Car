ESP32 WiFi Controlled RC Car
  A fully-featured WiFi-controlled RC car built using ESP32, PWM motor control, OTA updates, and a simple web interface.

Features:
  ESP32 Web Server for controlling the car
  Works in Station mode (WiFi) or Access Point mode
  Smooth motor speed control via LEDC PWM
  OTA Update support
  Control using any smartphone browser

Hardware Required
  ESP32 Dev Module
  L298N Motor Driver
  4 Gear Motors
  12V Battery
  Jumper wires
  Wheels
